Vipth's Scripts

These scripts help automate installation and updates for my home server.

Jumpstart

Ubuntu

Runs my essential commands for fresh installs.

wget -qO- vipth.app/jumpstart | sudo bash

Update Crafty

Docker

Updates Crafty, a Minecraft server management tool.

wget -qO- vipth.app/updatecrafty | sudo bash

Get Docker

Ubuntu

Installs Docker on your system.

wget -qO- vipth.app/getdocker | sudo bash
