Vipth's Scripts

These scripts help automate the installation and updates for my home server.

Jumpstart

Runs my essential commands for fresh Ubuntu installs.

wget -qO- vipth.app/jumpstart | sudo bash

Update Crafty

Updates Crafty, a Minecraft server management tool. (Docker)

wget -qO- vipth.app/updatecrafty | sudo bash

Get Docker

Installs Docker on your (Ubuntu) system.

wget -qO- vipth.app/getdocker | sudo bash
